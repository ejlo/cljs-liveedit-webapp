# hello

A hello world as a ClojureScript web app.


## Dependencies

* ruby, gem
* nodejs, npm

## Usage

```bash
scripts/install_deps.sh
foreman start
firefox http://localhost:63770
```

## Features

* automatically run tests
* live reload of css and js files
